# face detect 
This project is the image process sdk which is researched by Hangzhou Qiantu Technology Copyright.

## Content
This sdk include the following features, most of these are realtime,
* Saturation
* Lightness
* HueAndSaturation
* BrightContrast
* ColorTemperature
* HightlightShadow
* Exposure
* GammaCorrect
* SharpenLaplace
* OverExposure
* Desature
* Mosaturate
* Mosaic
* VirtualFilter
* CalcWH
* ImageTransformation
* RGBA2BGRA
* BGRA2RGBA
* Image filter
* Beauty Skin
